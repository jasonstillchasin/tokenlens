# Roadmap

Explicitly out of scope for v1 — noted here so they don't sneak back in:

- Live API proxy
- Web dashboard
- Cursor / OpenCode support
- Automatic config editing (e.g. auto-disconnecting MCP servers)
- A hosted service
- Telemetry

If one of these becomes worth doing, it gets its own milestone with its own
acceptance check, not a quiet expansion of an existing command.
